# Online Exam Management System

A simple **Java console-based application** that simulates an **online exam environment** for students and teachers. This project demonstrates **Object-Oriented Programming (OOP) concepts** such as **Encapsulation, Inheritance, Polymorphism, and Abstraction**.

## Features
* **User Management**: Students and Teachers can register, login, and view their information.
* **Exam Management**: Teachers can create exams; students can view and take exams.
* **Demo Data** included for testing:
  * Students: `S001` (1234), `S002` (abcd)
  * Teacher: `T001` (admin)
  * Sample Exam: `Math Midterm`
## Usage
1. Login as **Student** or **Teacher**.
2. Students can view exams and take them.
3. Teachers can create new exams and view all exams.
